Project Overview

This project involves the design and construction of a four-legged autonomous walking robot inspired by the locomotion of real-world quadruped robots. The robot (will) feature 3D-printed legs, MG996R servo motors at each joint, and a custom- designed chassis with a central spine. The goal of the project is to achieve stable, coordinated leg movement across multiple gaits, progressing from basic walking to more dynamic motion patterns.
